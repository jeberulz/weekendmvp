## 3 Lies About Startup Ideas

### Lie #1: "You need a unique, billion-dollar idea"

No, you don't. You need a **$1k/month idea** first. Something that solves a real pain for a specific person. Something you can actually build. The billion-dollar stuff comes later (if ever). Start small.

### Lie #2: "You need to validate for months before building"

Validation paralysis is real. You can spend 3 months surveying people, building spreadsheets, and running "market research." Or you can **ship a demo in 48 hours** and get real feedback from real people. Which one teaches you more?

### Lie #3: "You need a technical co-founder"

Maybe in 2015. Today you have Claude, Cursor, Replit, and a dozen AI tools that can help you build a working demo. You don't need a co-founder. You need **a weekend and the right prompts**.
